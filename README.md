# Sales Data Analysis Using Excel

## Introduction
This project focuses on analyzing sales data using Microsoft Excel to extract meaningful insights such as total revenue, sales trends, and best-selling products.

## Objectives
- Analyze sales data efficiently  
- Calculate total and average revenue  
- Identify top-performing products  
- Visualize data using charts and graphs  

## Tools Used
- Microsoft Excel  
- Excel Functions (SUM, AVERAGE, IF, COUNT)  
- Charts (Bar Chart, Pie Chart, Line Graph)  

## Sample Dataset

| Product Name | Quantity Sold | Price (₹) | Total Revenue (₹) |
|--------------|--------------|----------|-------------------|
| Product A    | 50           | 200      | 10000             |
| Product B    | 30           | 150      | 4500              |
| Product C    | 20           | 300      | 6000              |

Formula used:  
`Total Revenue = Quantity Sold × Price`

## Steps Involved
1. Enter and organize sales data in Excel  
2. Apply formulas to calculate revenue and totals  
3. Create charts (Bar, Pie, Line)  
4. Analyze patterns and trends  

## Key Insights
- Product A generated the highest revenue  
- Sales trends can be identified using line charts  
- Visualization helps in quick decision-making  

## Conclusion
Microsoft Excel is a powerful and user-friendly tool for analyzing sales data. It helps businesses understand performance, identify trends, and make data-driven decisions.
